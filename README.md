# Simulasi_Papan_Score_Basket
Simulasi Program untuk perhitungan score pada pertandingan bola basket dengan menggunakan ATEMEGA 128 Proteus dan AVR.

Software :
1. Software Proteus 			                 		
2. Software Codevision AVR						

Hardware :
1. Atmega 128.										
2. RES											
3. 7SEG-MPX1-CC
4. 7SEG-MPX2-CC
5. BUTTON
6. Ground
7. Power
8. BUZZER
